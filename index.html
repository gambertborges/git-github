Git e GitHub

Entenda de forma simples o que são e suas diferenças

O que é Git?

Git é um sistema de controle de versão que permite acompanhar mudanças no código ao longo do tempo. Ele ajuda desenvolvedores a trabalhar em equipe, salvar versões e voltar para versões anteriores do projeto.

O que é GitHub?

GitHub é uma plataforma online que utiliza o Git para armazenar projetos. Ele permite compartilhar código, colaborar com outras pessoas e acessar projetos de qualquer lugar com internet.

Diferenças

Git GitHub
Software local Plataforma online
Controla versões Armazena e compartilha
Funciona offline Precisa de internet
